# Product Naming: Mnemora

## Final Selection

| Name        | Etymology                                     | Meaning                                               |
| :---------- | :-------------------------------------------- | :---------------------------------------------------- |
| **Mnemora** | mnem- (Greek "memory") + ora (Latin "moment") | Remember the moment, implies real-time screen capture |

**Slogan:** **"Every moment, remembered."**
